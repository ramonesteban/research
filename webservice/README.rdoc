== README

=== Requirements
* Ruby version <tt>2.0</tt>

* Rails version <tt>4.0</tt>


=== Configurations
* Install all gems dependencies:

    $ bundle install

* To create the database run:

    $ bundle exec rake db:migrate

* To start the server in development mode, run:

    $ bundle exec rails s

=== Open the application
Once you have started the server, open your web browser at:

    http://localhost:3000/
